# Lab 07 - Prompt Engineering Workshop

Module: 7 - Generative AI

Dataset:

Not required.

Reference:

https://github.com/0xKatie/ai-ml-mastery-hub

## Goal

Teach learners how to write clear prompts for generative AI tools.

## Trainer Explanation

A good prompt gives:

- Role.
- Task.
- Context.
- Format.
- Constraints.
- Example, if needed.

## Learner Task

Improve this weak prompt:

```text
Write about AI.
```

Better prompt:

```text
Act as a training instructor. Explain artificial intelligence to beginners in 5 bullet points. Use simple examples from banking, education, and healthcare. Avoid technical jargon.
```

## Practice Activities

Ask learners to create prompts for:

1. Summarizing a long article.
2. Creating a lesson plan.
3. Generating Python code.
4. Explaining a machine learning result.
5. Writing an email to a customer.

## Expected Output

Learners should produce prompts that are specific, structured, and easy for an AI tool to follow.

## Trainer Tip

Show the same task with a weak prompt and a strong prompt. Compare the answers.
