# Lab 01 - Industry Use Case Analysis

Module: 1 - Intro to AI

Dataset: Not required. This is a conceptual lab.

## Goal

Help learners understand where AI is used in real industries and how to identify an AI use case.

## Trainer Explanation

AI is useful when a system needs to learn from data, identify patterns, make predictions, recommend actions, or generate content.

Examples:

- Banking: fraud detection.
- Healthcare: disease risk prediction.
- Retail: product recommendation.
- Education: student performance prediction.
- Manufacturing: defect detection.
